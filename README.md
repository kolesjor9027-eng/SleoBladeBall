# SLEO Blade Ball Mod Menu

A feature-rich mod menu for Blade Ball with PC & Mobile support.

## Features

### Main Tab
- **Auto Parry** - Automatically detects and counters all abilities
- **Auto Jump** - Automatically jumps continuously
- **Speed Changer** - Adjust your walk speed (1-5000)

### Trade Tab
- **Freeze Trade** - Freezes the trade GUI
- **Force Accept** - Automatically accepts trades

## Usage

### Loadstring (Roblox Executor)
Copy and paste this into your Roblox executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/kolesjor9027-eng/SleoBladeBall/main/main.lua", true))()
```

## Requirements

- Roblox Game Executor (Synapse X, Script-Ware, etc.)
- Works on both **PC** and **Mobile**
- Blade Ball game

## Notes

- The UI automatically scales for mobile devices
- All features can be toggled on/off with the ON/OFF buttons
- Speed Changer accepts values from 1 to 5000
- Draggable menu - click and drag the title bar to move
